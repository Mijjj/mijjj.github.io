教你通过软件创建HTTP代理并连接

2016-11-19 网络 ， 工具

2018.11.10 打算写 Clash ，最近吧（想起来再写2020.1.12）

简要

本文章教你利用软件创建HTTP代理并使其他设备连接代理服务。教程包含Surge2＆3、Quantumult、Shadowrocket和某R的HTTP代理创建，以及iOS、Win10和Nintendo Switch的HTTP代理连接设置。 # 创建HTTP代理服务 下面会列举几个能够创建HTTP代理服务的软件使用方法。包含Surge2＆3、Quantumult、Shadowrocket、某R（电脑端）、某RR（安卓端）。（其中除了某R和某RR外，其余均为iOS中的app，且在中区均下载/购买不到）

Surge3

Surge 3 - Web Developer Tool by Yachen Liu https://itunes.apple.com/cn/app/surge-3-web-developer-tool/id1329879957?l=en&mt=8

  1. 进入Surge，开启开关并选择规则模式，在首页下方点击进入“高级设置” 这里状态栏显示“4G”为iOS11的BUG，实际上我已经连接了Wifi

  2. 打开“允许 Wi-Fi 访问”的开关，点击完成

  3. 返回首页点击左上角的“配置名”进入配置列表

  4. 点击“在文本模式中编辑“

  5. 找到［port = xxxxx］字段，其中的“xxxxx”几个数字便是端口，若实在找不到，软件默认设置为6152

  6. 「代理地址」启动Surge设备的内网IP

Quantumult

Quantumult by Qian Hua https://itunes.apple.com/us/app/quantumult/id1252015438?mt=8 它是目前在iOS中区仍仅存活的工具

  1. 进入软件的“More”界面，点击“Accessibility”

  2. 打开“Allow Wi-Fi Access”的开关

  3. 回到“Home”界面，打开开关，你可以在软件左上角看到「代理地址」以及端口

Shadowrocket

Shadowrocket 作者是 Guangming Li https://appsto.re/cn/UDjM3.i

  1. 打开你的Shadowrocket，点开设置页面。

  2. 滚到下面找到“代理”，点开

  3. 点开“代理共享”，打开“启用共享”

Surge2

Surge2 作者是 Yachen Liu [https://itunes.apple.com/us/app/surge-legacy-support/id1040100637?mt=8](https://itunes.apple.com/us/app/surge-legacy-support/id1040100637?mt=8

)

  1. 打开Surge，进入到“Config”配置界面

  2. 进入Edit界面后，拖到下面点击“Advance”。

  3. 点开开关

  4. 来到text Edit mode（即Configuration下面那个），找到［port = xxxx］字段，其中的“xxxx”几个数字便是端口，若实在找不到，软件默认设置为6152

  5. Surge的「代理地址」就是开着Surge的设备的内网IP

https://github.com/shadowsocksrr/shadowsocksr-android/releases

  3. 其「代理地址」为此电脑在内网中的IP地址

最后

如果在上述的设置过后，你仍然无法翻越墙壁，先检查你的代理服务有无配置正确，再检查自己上述操作有无问题。（一般是没有的）

ps：提供代理服务的设备必须时刻内网中，若此设备脱离了内网（离开了Wifi环境），就必须要关闭其他设备的HTTP代理，否则无法上网。
